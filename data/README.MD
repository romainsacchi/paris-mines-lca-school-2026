# Course Data

This folder contains bundled data used during the Paris course.

Included content:
- `lci-bafu.xlsx`: the Brightway-compatible Excel version of the BAFU:2025 inventory used in Day 1 and Day 4
- `LCIA methods/`: importable Excel LCIA workbooks kept as reusable teaching assets

Notes:
- Participants do not need to download the BAFU database before the course.
- The BAFU website link elsewhere in the repository is provided only for reference and terms of use.
- `ecoinvent` is not included in this repository, but it is used later in the course after participants import it locally with their own credentials.
- Participants do not need to rename, move, or edit these files before the course unless a notebook explicitly asks them to.
