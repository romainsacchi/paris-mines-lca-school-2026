# Tutorials Index

Core course structure:

- `00_SETUP/`: run before Day 1 to verify installation, imports, and the active kernel
- `DAY 1 - Brightway Foundations/`: builds the core Brightway project state with the BAFU workbook
- `DAY 2 - Brightway and Activity Browser Part I/`: imports `ecoinvent 3.12 cutoff` and switches into the first GUI module
- `DAY 3 - Premise and Activity Browser Part II/`: generates and compares prospective databases
- `DAY 4 - Edges and Trails/`: runs the compact `edges` sequence and optional `trails` material
- `OPTIONAL/`: advanced or reference notebooks kept outside the main teaching path

Use [SCHEDULE.MD](../SCHEDULE.MD) for the timed agenda and notebook order.
